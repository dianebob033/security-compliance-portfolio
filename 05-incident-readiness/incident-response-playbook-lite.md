# Incident Response Playbook (Lite)

## Goals
Contain and recover from security incidents with clear roles and steps.

## Severity Levels
- SEV1 Critical / SEV2 High / SEV3 Medium / SEV4 Low

## Roles
- Incident Commander
- Security Lead
- Comms / Stakeholders
- Engineering Support

## Core Steps
1. Detect & Triage
2. Contain
3. Eradicate
4. Recover
5. Post-incident review (lessons learned)

## Evidence
- Incident ticket template
- Timeline log
